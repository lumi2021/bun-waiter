# BUN WAITER

`bun waiter` is a framework developed to creating easiily and complete REST APIs with Bun.
It structure is based on the standard ASP.NET Web API.

bun waiter features includes:
- graphic user interface;
- Cors;
- Routing;
- Websockets;
- (more in the future)
